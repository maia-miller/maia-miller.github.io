Blog 4 - Technical Blog P2 Reflection

  • What is grid based design?

    • Grid-based design is sort of like boxes on websites, in that it's ubiquitous and foundational to all websites. Grid based design is a way to organize the information on a page to follow the way that we naturally read a page - left to right, and top to bottom.

  • Whats all the hype about responsive webpages?

    • Responsive webpages are really important to make sure that your page can be read on every type of screen. Since today people are browsing the internet on all sorts of devices with all sorts of screen sizes (from gigantic monitors to tiny cellphones), it's important that a developer makes their work look good and readable on all screen sizes.

  • What is semantic structure?

    • These are the tags that can be easily read by scanning the code. It includes tags like headings, img, links (a href), etc. They're semantic structure because they're more part of our everyday language.
