Blog 12 - JS Language

In the JavaScript language, describe all the use cases for:

  • parentheses ( )

    • Parentheses offer the input for loops or functions.

    1. Specifying the conditions for an if loop: if (condition) {};
    2. Outline the parameters of a function: function thisCommand (param1, param2) {};
    3. Call upon a function with a given argument: thisCommand(argument);
    4. Grouping many things, like in math equations: (.isMine == true) && (.isMarked == true);
    5. Calling upon a method: Math.floor(arg);
    6. Defining a for loop: for (i = 0; i < length; i++) {};
    7. Defining a while loop: while (condition){};
    8. Defining the list through which to run a switch: swith(thisList) {};
    9. Add an event listener: one.addEventListener(event, functionName);

  • brackets [ ]

    • Brackets call upon an item in an array or index. They also define the items in an array.

    1. Identify the items within an array: theArray = [item1, item2, item3];
    2. Call a specific item in an array: theArray[2]
    3. Within a for loop, use the i variable that you have defined: for (i = 0; i < length; i++) {if (board[i] === true) {return}}
    4. A notation option for calling an item within an array or object (particularly useful if the item is a string): myStorage.cabinet["top drawer"].folder2

  • braces { }

    • Braces are used to define the code to be applied to an item or data. It is also used to define the items of an object.

    1. When defining the code block to be applied to a for loop: for (i =<++) {do this stuff};
    2. When defining the code block to be applied to a if or while loop: if/while (condition is met) {then do this};
    3. When defining the code block to be applied to a function: function functionName (param1) {code here};
    4. When defining the code block to be applied to a switch case: switch (arg) {case 1: code; break; case 2: codebreak;};
    5. When defining the code block to be applied to a method: myList.forEach(item) {do this code};
    6. To define the items of an object: var object = {name: Maia, location: Toronto};

  • single quotes ' ' and double quotes " "

    • Can be used interchangeably, so long as they are used consistently (don't start with single and finish the phrase with double). Qutoes indicate a string: console.log("Hello world!");
